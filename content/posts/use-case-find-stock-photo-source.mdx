# Use Case: Finding Stock Photo Sources

Identifying stock photo sources helps you understand licensing and usage rights. This use case shows you how to find stock photo origins.

## Why Find Stock Photo Sources?

Finding sources helps you:
- Understand licensing terms
- Purchase proper licenses
- Verify usage rights
- Find photographers
- Get high-resolution versions

## Process

1. **Reverse Search**: Upload to [reverse image search tool](/tools/reverse-image-search-links)
2. **Check Stock Sites**: Look for stock photo platform results
3. **Verify Licensing**: Check license types and terms
4. **Purchase if Needed**: Buy proper license
5. **Attribute Properly**: Follow attribution requirements

## Common Sources

- Shutterstock
- Getty Images
- Adobe Stock
- Unsplash
- Pexels

## Best Practices

- Always verify licensing
- Check usage terms
- Purchase when needed
- Attribute properly
- Respect copyright
