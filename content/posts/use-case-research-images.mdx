# Use Case: Researching Images

Using reverse image search for research helps verify sources, check facts, and gather information. This guide covers research applications.

## Research Applications

Research uses include:
- Verifying image sources
- Fact-checking claims
- Finding original context
- Gathering information
- Academic citations

## Research Process

1. **Upload Image**: Use [reverse image search tool](/tools/reverse-image-search-links)
2. **Find Sources**: Locate original sources
3. **Verify Information**: Check facts and context
4. **Document Sources**: Save citations
5. **Cross-Reference**: Verify with multiple sources

## Best Practices

- Verify all sources
- Check publication dates
- Cross-reference findings
- Document everything
- Cite properly
