# Use Case: Checking for Copyright Violations

Monitoring your images for unauthorized use is essential for protecting your work. This guide shows you how to check for copyright violations.

## Why Check for Violations?

Checking helps you:
- Protect your work
- Find unauthorized use
- Enforce copyright
- Monitor your images
- Take legal action if needed

## Checking for Violations

1. **Regular Searches**: Use [reverse image search tool](/tools/reverse-image-search-links) regularly
2. **Check Results**: Review where images appear
3. **Verify Usage**: Check if use is authorized
4. **Document Findings**: Save evidence
5. **Take Action**: Contact infringers or file DMCA

## Best Practices

- Search regularly
- Check multiple providers
- Document everything
- Contact professionally
- Enforce when needed
