# Use Case: Monitoring Images on Social Media

Monitoring your images on social media helps protect your brand and track content usage. This guide shows you how to monitor social platforms.

## Why Monitor Social Media?

Monitoring helps you:
- Protect your brand
- Track content usage
- Find unauthorized use
- Monitor engagement
- Enforce copyright

## Monitoring Process

1. **Regular Searches**: Use [reverse image search tool](/tools/reverse-image-search-links) weekly
2. **Check Platforms**: Look for social media results
3. **Review Usage**: Verify if use is authorized
4. **Document Findings**: Save screenshots and links
5. **Take Action**: Contact or report if needed

## Key Platforms

- Instagram
- Facebook
- Twitter/X
- Pinterest
- TikTok

## Best Practices

- Search regularly
- Check all platforms
- Document violations
- Contact professionally
- Use platform reporting
