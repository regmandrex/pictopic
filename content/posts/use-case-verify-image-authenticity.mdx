# Use Case: Verifying Image Authenticity

Verifying image authenticity is crucial in an age of digital manipulation. This guide shows you how to fact-check images.

## Why Verify Images?

Verification helps you:
- Avoid spreading misinformation
- Verify news claims
- Check social media posts
- Verify product images
- Fact-check articles

## Verification Process

1. **Reverse Search**: Use [reverse image search tool](/tools/reverse-image-search-links)
2. **Check Sources**: Find original sources
3. **Verify Dates**: Check publication dates
4. **Cross-Reference**: Verify with other sources
5. **Check Context**: Ensure context matches

## Red Flags

- No original source found
- Recent upload dates for old claims
- Multiple conflicting stories
- Suspicious metadata
- Manipulation signs

## Tools

- Reverse image search tools
- Fact-checking websites
- Metadata analyzers
- Multiple search providers
